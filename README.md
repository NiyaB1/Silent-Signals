# Silent-Signals
AI System to Detect Early Burnout &amp; Stress Using Everyday Keyboard &amp; Mouse Behavior

# What it does:
- Collects typing speed, pause patterns, error rate, mouse movement
- Extracts features over time windows
- Detects deviation from personal baseline
- Flags early burnout risk trends, not labels

# Component	Tool
- Data collection:	Python (pynput)
- Feature extraction:	Python (NumPy, Pandas)
- ML:	Scikit-learn (KNN / Isolation Forest)
- Visualization:	Matplotlib
- Interface:	CLI
